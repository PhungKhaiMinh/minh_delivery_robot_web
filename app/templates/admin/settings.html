Chế độ điều khiển

Auto: robot chạy theo lộ trình / stack điều khiển của bạn. Manual: dùng tốc độ bánh và calib IMU bên dưới.

Tốc độ bánh (chỉ khi Manual)

Đơn vị do firmware quy ước (ví dụ RPM, duty %, m/s). Nhập số âm nếu robot hỗ trợ lùi.

Calib góc Yaw (IMU)

Gửi giá trị offset góc yaw lên UGV/offset/calib_heading — firmware dùng giá trị này để calib heading IMU.

Payload JSON mẫu 
// UGV/Control
{
  "manual_mode": true,
  "left_vel": 120,
  "right_vel": 118
}

// UGV/offset/calib_heading
{
  "calib": 0.157
}
